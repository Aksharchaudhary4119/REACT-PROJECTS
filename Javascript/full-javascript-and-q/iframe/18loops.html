if...else statements

      if (condition) {
        /* code to run if condition is true */
      } else {
        /* run some other code instead */
      }

else if

The last example provided us with two choices, or outcomes — but what if we want more than two?

There is a way to chain on extra choices/outcomes to your if...else — using else if. Each extra choice requires an additional block to put in between if () { } and else { } — check out the following more involved example, which could be part of a simple weather forecast application:

        const select = document.querySelector("select");
        const para = document.querySelector("p");
        
  select.addEventListener("change", setWeather);
  
  function setWeather() {
    const choice = select.value;
    
    if (choice === "sunny") {
      para.textContent =
      "It is nice and sunny outside today. Wear shorts! Go to the beach, or the park, and get an ice cream.";
    } else if (choice === "rainy") {
      para.textContent =
      "Rain is falling outside; take a rain coat and an umbrella, and don't stay out for too long.";
    } else if (choice === "snowing") {
    para.textContent =
    "The snow is coming down — it is freezing! Best to stay in with a cup of hot chocolate, or go build a snowman.";
  } else if (choice === "overcast") {
    para.textContent =
    "It isn't raining, but the sky is grey and gloomy; it could turn any minute, so take a rain coat just in case.";
  } else {
    para.textContent = "";
  }
}
Question And Answer

1. difference between exec () and test () methods

In JavaScript, both exec() and test() methods are used with regular expressions, but they serve different purposes and return different types of results.

'exec()' Method:

  • The exec() method is a method of the RegExp object in JavaScript.
  • It is used to search for a match in a string based on the regular expression pattern provided.
  • If a match is found, exec() returns an array containing information about the match, including the matched text and captured groups. If no match is found, it returns null.
  • If the regular expression is global (has the g flag), subsequent calls to exec() will continue searching for matches in the string, starting from where the last match ended.
      const regex = /hello (\w+)/;
      const str = 'hello world';
      const result = regex.exec(str);
      console.log(result); // Output: ["hello world", "world"]

2.test() Method:

  • The test() method is a method of the RegExp object in JavaScript.
  • It is used to test whether a string contains a match for the regular expression pattern provided.
  • Unlike exec(), test() returns a boolean value (true or false) indicating whether a match is found in the string.
  • It does not provide detailed information about the match, such as the matched text or captured groups; it only indicates whether there is a match
      const regex = /hello/;
      const str = 'hello world';
      const result = regex.test(str);
      console.log(result); // Output: true
