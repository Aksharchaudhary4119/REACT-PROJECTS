Question and Answer

1. How do you create an object in JavaScript?

  • Create a single object, using an object literal.
  • Create a single object, with the keyword new .
  • Define an object constructor, and then create objects of the constructed type.
  • Create an object using Object.create() .

2. What is the ‘this’ keyword in JavaScript?

“This” keyword refers to an object that is executing the current piece of code. It references the object that is executing the current function. If the function being referenced is a regular function, “this” references the global object.

3. What are object prototypes?

Prototypes are the mechanism by which JavaScript objects inherit features from one another.

4. Use of a constructor function (with examples)

//Constructor function User (name, age) { this.name = name; this. age = age; } var user1 = new User('Bob', 25); var user2 = new User('Alice', 27); In the above example, arguments are passed to the constructor during object creation. This allows each object to have different property values.

5. Prototypal vs Classical Inheritance

In classical inheritance, Animal is a class, Dog is a subclass of Animal , and suzanna is an instance of a Dog . In prototypal inheritance, there's no class. You have an animal , which is an object. A dog is another object, which clones and extends animal (the prototype object).
